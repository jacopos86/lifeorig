# Volcanic rock environment reaction network v1
# Purpose:
#   Environment-specific reference network for subaerial volcanic fields,
#   basaltic glass, ash beds, lava tubes, fumarolic margins, hot-spring edges,
#   and porous volcanic-rock films exposed to wet-dry, gas-rock, and thermal
#   cycling.
#
# Scope:
#   This network is distinct from hydrothermal vent and surface pond networks.
#   It emphasizes volcanic gas deposition, basalt/glass dissolution, mineral
#   surface catalysis, sulfur/iron redox, dry-down concentration on rock pores,
#   thermal activation, phosphate release/trapping, peptide/lipid condensation,
#   and intermittent aqueous transport.
#
# Status:
#   Literature-guided hypothesis network for comparative simulations.
#   Several entries are effective/lumped processes designed for network studies.
#
# Columns:
#   ID | MODULE | REACTION | CATALYST_OR_CONTROL | RATE_TEMPLATE | ROLE | REFS | CONFIDENCE
#
# Environmental variables:
#   T_surface, T_pore, pH_film, water_activity, wet_dry_cycle_phase, gas_flux,
#   UV_flux, redox_state, mineral_SA[basalt_glass,olivine,pyroxene,magnetite,
#   FeS,NiS,clay,apatite], film_thickness, evaporation_rate, rainfall_rate.
#
