# Surface pond reaction network v1
# Purpose:
#   Environment-specific reference network for shallow prebiotic surface ponds,
#   lakeshores, evaporating pools, ice/eutectic ponds, and UV-irradiated wet-dry
#   settings on early Earth-like rocky planets.
#
# Scope:
#   This network is distinct from hydrothermal vent and volcanic rock networks.
#   It emphasizes atmospheric delivery, UV photochemistry, HCN concentration,
#   formamide/nitrile chemistry, formose and Strecker pathways, wet-dry
#   polymerization, phosphate activation, amphiphile self-assembly, and
#   photochemical/evaporative losses.
#
# Status:
#   Literature-guided hypothesis network for comparative simulations.
#   Some entries are lumped effective reactions or rule templates.
#
# Columns:
#   ID | MODULE | REACTION | CATALYST_OR_CONTROL | RATE_TEMPLATE | ROLE | REFS | CONFIDENCE
#
# Environmental variables:
#   T, pH, ionic_strength, UV_flux(lambda), wet_dry_cycle_phase,
#   evaporation_factor, freezing_factor, water_activity, rainfall/dilution,
#   mineral_SA[clay,borate,phosphate,carbonate,FeS], atmospheric_flux[HCN,H2CO],
#   dissolved concentrations and activities.
