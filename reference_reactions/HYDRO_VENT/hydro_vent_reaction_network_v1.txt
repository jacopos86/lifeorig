# Hydrothermal vent reaction network v1
# Purpose:
#   Environment-specific reference network for alkaline/reducing ocean-rock
#   interfaces, hydrothermal pores, and vent-ocean mixing zones.
#
# Scope:
#   This network is distinct from surface pond and volcanic rock networks.
#   It emphasizes high-pressure aqueous speciation, serpentinization products,
#   Fe/Ni sulfide and magnetite surfaces, C1 reduction, thioester/acetyl
#   activation, reductive amination, peptide/lipid survival, and open-flow loss.
#
# Status:
#   Literature-guided hypothesis network for comparative simulations.
#   Many reactions are lumped effective pathways, not elementary mechanisms.
#

#   ID | MODULE | REACTION | CATALYST_OR_CONTROL | RATE_TEMPLATE | ROLE | REFS | CONFIDENCE
