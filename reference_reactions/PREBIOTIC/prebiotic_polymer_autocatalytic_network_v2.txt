# Prebiotic polymer/autocatalytic reaction network v2
# Purpose:
#   Rule-based, environment-coupled network for testing whether planetary
#   feedstocks can generate catalytic polymers and autocatalytic closure.
#
# Status:
#   Hypothesis network. It is not a validated kinetic mechanism.
#   It is designed to be computationally useful: each reaction/rule has
#   reactants, products, catalysts, conditions, and an environment-dependent
#   rate law template.
#
# Environmental variables expected from atmosphere/geochemistry:
#   T                 temperature [K]
#   P                 pressure [Pa]
#   pH                aqueous pH from PHREEQC
#   I                 ionic strength from PHREEQC
#   UV                local UV photon/photolysis intensity
#   redox             redox proxy, e.g. H2/CO2, Fe2+/Fe3+, HS-/SO4
#   aw                water activity or wet/dry concentration factor
#   S_mineral[X]      reactive surface area for mineral X
#   C[X]              concentration/activity of species X
#
# General rate-law templates:
#   Arrhenius:
#       k_T = A * exp(-Ea / (R*T))
#
#   pH window:
#       f_pH = exp(-((pH - pH_opt)/sigma_pH)^2)
#
#   acid/base catalysis:
#       f_acid_base = 1 + aH * 10^(-pH) + aOH * 10^(pH-14)
#
#   mineral catalysis:
#       f_min = 1 + alpha_min * S_mineral[mineral]
#
#   polymer catalysis:
#       f_poly = 1 + alpha_poly(N, sequence_class) * C[polymer]
#
#   wet/dry concentration:
#       f_wetdry = aw_factor or cycle_factor
#
#   photolysis:
#       rate = J_i(T,P,z) * C[i]
#
#   generic catalyzed mass action:
#       rate = k_T * f_pH * f_I * f_condition * product(C[reactants])
#            * (1 + sum_j alpha_j * C[catalyst_j])
#
# Species classes:
#   FEEDSTOCKS:
#       HCN, H2CO, NH3, CO, CO2, H2, H2S, HS-, CH3SH, phosphate, H2O
#
#   SMALL ORGANICS:
#       formamide, formate, acetate, glyoxylate, glycolaldehyde, glyceraldehyde,
#       pyruvate, oxaloacetate, succinate, malate, fumarate, alpha_ketoglutarate
#
#   AMINO ACID PRECURSORS:
#       glycinonitrile, alaninonitrile, aspartate_nitrile
#
#   AMINO ACIDS:
#       glycine, alanine, aspartate, glutamate
#
#   ENERGY / ACTIVATED SPECIES:
#       thioacetate, acetyl_phosphate, aminoacyl_thioester[AA],
#       aminoacyl_phosphate[AA]
#
#   POLYMERS:
#       pep[sequence]               peptide oligomer
#       hcn_oligo[n]                HCN oligomer
#       formose_oligo[n]            sugar/formaldehyde oligomer
#       hetero_oligo[n]             mixed N/O/C organic oligomer
#
#   MINERALS / SURFACES:
#       FeS, NiS, FeOOH, clay, phosphate_surface, mineral_polymer_complex
#
# Condition sets:
#   K01_ATMOS_PHOTOLYSIS:
#       gas/aerosol/surface interface; UV present; uses atmospheric UV attenuation.
#
#   K02_HCN_POND:
#       concentrated HCN/NH3/H2CO in surface pond/eutectic/drying environment;
#       T=273-350 K; pH neutral-alkaline; wet/dry or freezing concentration.
#
#   K03_FORMOSE_ALKALINE:
#       H2CO-rich pond; pH alkaline; divalent ions/minerals; T=273-350 K.
#
#   K04_STRECKER_AQUEOUS:
#       aldehydes + HCN + NH3; moderate pH; T=273-373 K.
#
#   K05_THIOESTER_HYDROTHERMAL:
#       CO/CO2 + H2/HS-/CH3SH + FeS/NiS; reducing; T=300-650 K.
#
#   K06_WET_DRY_POLYMERIZATION:
#       surface pond / volcanic rock / drying mineral surface; T=280-380 K.
#
#   K07_HYDROTHERMAL_CARBOXYLATE:
#       ocean-rock interface; high P; FeS/NiS; CO2/CO/H2; T=350-650 K.
#
#   K08_POLYMER_CATALYSIS_TEST:
#       kinetic box for autocatalytic threshold tests; polymer catalysts active
#       with probability p_cat(N, class).
#

#    are generative rules. Expand them into concrete reactions for the selected
#    alphabet and maximum polymer length.

# 2. Catalysis probability must be computed from catalyst class and size:
#       p_cat = p0_class * size_factor(n) * environment_factor(T,pH,I,mineral)
#
# 3. Autocatalysis should be measured from the generated network topology:
#       - food set supplied by atmosphere/geochemistry
#       - reactions enabled by catalysts produced inside the network
#       - growth condition: production flux of catalyst class exceeds loss flux
#
# 4. The network should be filtered by environment:
#       surface_pond      -> K02, K03, K04, K06, K08
#       hydrothermal_vent -> K05, K07, K08
#       volcanic_rock     -> K02, K04, K06, K08
#
# 5. Rate constants are not assigned here. This file defines the symbolic
#    reaction/rule structure and the environmental dependencies needed for
#    rate construction.
